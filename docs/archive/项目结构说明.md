# 问卷星自动填写系统 - 项目结构说明

## 📁 **项目目录结构**

### 🔥 **根目录核心文件**
```
wjx-master/
├── 问卷星终极版.py          # 🎯 主程序入口
├── 启动系统.py               # 🚀 系统启动脚本
├── chromedriver.exe          # 🌐 Chrome浏览器驱动
├── config.json               # ⚙️ 系统配置文件
├── README.md                 # 📖 项目说明文档
├── 使用说明.md               # 📋 用户使用指南
├── 配置参数说明.md           # 🔧 配置参数详解
├── 问卷解析修复完成指南.md   # 🎉 修复完成指南
├── 问卷星终极版.spec         # 📦 PyInstaller打包配置
└── submission_log.csv        # 📊 提交记录日志
```

### 📂 **主要目录结构**

#### `config/` - 配置管理
```
config/
├── profiles/
│   └── sample_answers.json  # 示例答案模板
└── settings/
    ├── config_manager.py    # 配置管理器
    └── logger_config.py     # 日志配置

#### `core/` - 核心功能模块
```
core/
├── ai/
│   ├── ai_chat_tab.py              # AI聊天界面

│   └── questionnaire_filler.py     # 问卷填写器
└── parser/
    └── questionnaire_parser.py     # 问卷解析器

#### `ui/` - 用户界面组件
```
ui/

```

#### `scripts/` - 脚本工具
```
scripts/
├── 快速启动.bat           # 一键启动脚本

#### `docs/` - 文档和示例
```
docs/

#### `tests/` - 测试文件
```
tests/

#### `data/` - 数据目录
```
data/

logs/

```
resources/

#### `dist/` - 发布文件
```
dist/

## 🎯 **使用流程**

### 1. 快速启动
```bash
# 方法1：双击批处理文件
scripts\快速启动.bat

# 方法2：使用Python直接运行
python 问卷星终极版.py

# 方法3：使用启动脚本
python 启动系统.py
```

### 2. 开发调试
```bash
# 调试模式启动
scripts\调试启动.bat

# 测试解析功能
scripts\快速测试解析.bat

# 运行测试套件
python -m pytest tests/
```

## 📋 **文件用途说明**

### 🔥 **必须保留的核心文件**
- `问卷星终极版.py` - 主程序，包含所有核心功能
- `chromedriver.exe` - Chrome浏览器驱动，用于自动化操作
- `config.json` - 系统配置文件，存储用户设置
- `core/` 目录 - 核心功能模块，不可删除
- `ui/` 目录 - 用户界面组件，UI相关功能

### 📁 **重要配置和数据**
- `config/` - 配置管理相关文件
- `data/` - 数据存储目录
- `logs/` - 日志文件，用于问题排查

### 🛠️ **开发和维护**
- `docs/` - 文档和参考实现
- `tests/` - 测试文件，确保功能正常
- `scripts/` - 便捷脚本，提高使用效率

### 📦 **可选文件**
- `dist/` - 编译后的可执行文件
- `resources/` - 资源文件和备用驱动
- `问卷星终极版.spec` - 打包配置文件

## ⚠️ **注意事项**

1. **不要删除** `core/`、`ui/`、`config/` 目录中的文件
2. **Chrome驱动** 需要与浏览器版本匹配
3. **日志文件** 可以定期清理，但建议保留最近的
4. **配置文件** 请备份后再修改
5. **测试文件** 用于验证功能，建议保留
